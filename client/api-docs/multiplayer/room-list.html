GET qbreader.org/api/multiplayer/room-list

Get a list of public rooms.

Parameters

This endpoint takes no parameters.

Returns
A JSON object with the following properties:
  • roomList: Object[]

    An array of objects with the following properties:

    {
    roomName: string
    playerCount: number
    onlineCount: number
    isPermanent: boolean
}
