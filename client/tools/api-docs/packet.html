GET qbreader.org/api/packet

Get a specific packet from a set.

Parameters

 One of the following sets of parameters is required. Set 1:
  • setName: string required
    The name of the set. See /set-list for a list of all sets.
  • packetNumber: number required
    The number of the packet in the set, starting from 1.
If both an _id and a setName, packetNumber are provided, the _id will be used. Set 2:
  • _id: string required
    The _id of the set.

Optional Parameters
  • modaq: boolean default: "false"
    Whether or not to output in a format compatible with MODAQ.
  • questionTypes: ("tossups", "bonuses")[] default: ["tossups", "bonuses"]
    An array of question types to include in the response. If not provided, both tossups and bonuses will be included. Valid values are "tossups" and "bonuses".

Returns
A JSON object with the following properties:
  • tossups: Tossup[]
    An array of Tossups in the packet.
  • bonuses: Bonus[]
    An array of Bonuses in the packet.
  • packet: Packet
    A Packet object representing the requested packet.
