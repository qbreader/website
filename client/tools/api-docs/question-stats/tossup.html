GET qbreader.org/api/question-stats/tossup

Get aggregated QBReader play statistics for a given tossup.

Parameters
  • _id: string required
    The ID of the tossup.

Returns
A JSON object with the following properties:
  • _id: string
    The _id of the requested tossup.
  • count: number
    The number of times the tossup was played.
  • numCorrect: number
    The number of times the tossup was answered correctly.
  • pptu: number
    The average points earned on this tossup for each time it was played.
  • resultCounts: object
    For each key: value pair, key represents a possible number of points that a player could earn on the tossup, and value equals number of times the tossup was played and the player got points on the tossup equal to key.
    • Keys of 15, 10, 0, and -5 are guaranteed to be present.
    • The sum of all values of this object is guaranteed to equal count.
  • totalCelerity: number
    The total celerity earned on this tossup over every time it was played.
  • totalCorrectCelerity: number
    The total celerity earned on this tossup over every time it was answered correctly.
  • totalPoints: number
    The total points earned on this tossup over every time it was played.
