POST qbreader.org/api/report-question

Report a question error.

Parameters
  • _id: string required
    The internal ID of the question.
  • reason: string default: ""
    The reason why the question is being reported.
  • description: string default: ""
    A description of the problem.

Returns
200 OK if the operation was successful and 500 Internal Server Error if there was an error.
