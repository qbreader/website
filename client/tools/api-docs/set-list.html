GET qbreader.org/api/set-list

Get a list of all the sets in the database.

Parameters

This endpoint takes no parameters.

Returns
A JSON object with the following properties:
  • setList: string[]
    An array of all the sets in the database, sorted in reverse alphanumeric order.
