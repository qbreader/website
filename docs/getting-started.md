---
## Prerequisites
- [Node.js](https://nodejs.org/en) + npm
- MongoDB credentials for testing - Click [here](docs\CONTRIBUTING.md) for more info

---
## Installation
- make a new folder
- copy the path to the folder and run `cd YOUR_FOLDER_PATH` in your terminal
- run `git clone https://github.com/qbreader/website`

---
## Testing
- Check the [Contribution Guidlines](docs\CONTRIBUTING.md) to set up MongoDB
- Run `npm start` and open the website at localhost:3000
- For testing multiplayer features, open the 2 browser accounts and join the same room.
